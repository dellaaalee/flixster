


theme:
Dark, cinematic theme like netflix but still clean and minial design. Icons/ search is transparent. 


header
TheaterNow title/logo at the left most of the screen, browse tab to the next to the screen, and search next to the screen, the very right of the screen is a profile picture. The search will use assets/img/search_icon.png and will have a round input box with "search movie" as a default text. When you click on the TheaterNowlogo, it should take you to the TheaterNow page. 

When you are in a browse tab, the movie grid will populate using the api. It's animated to slide into the screen.


When the moviecard is clicked, the modal window will pop up. the poster image, title, runtime, vote average, release date, genre, and overview will be shown. It will also have "watch recommendation". 


When you click on the profile icon, it will have same design grid as the now playing page. On the top of the page, it will have <h1> with "continue watching..."



Now Playing endpoint returns 20 movies per page
